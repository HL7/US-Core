Title:US Core Imaging Result Observation Profile
Status:Active as of 11/9/21
Definition:

Defines constraints and extensions on the Observation resource for the minimal set of data to query and retrieve structured and unstructured (narrative) components of imaging test results. These observations include reasons, findings, and impressions.
