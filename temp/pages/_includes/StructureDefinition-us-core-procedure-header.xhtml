http://hl7.org/fhir/us/core/StructureDefinition/us-core-procedure

Defines constraints and extensions on the Procedure resource for the minimal set of data to query and retrieve patient's procedure information. This profile can be used to record a service or intervention that is or was performed on or for a patient.

This profile builds on Procedure.
