Name:USCoreObservationSocialHistoryAssessment
Title:US Core Social History Assessment Observation Profile
Status:Active as of 11/12/21
Definition:

Profile for simple observations made by an individual about a patient's social history status such as education, food insecurity, etc. These observation involve assessment by a provider, payer, etc. as opposed to being derived only from screening surveys which should be represented using the US Core Screening Response Observation Profile
