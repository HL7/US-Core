  • Include all codes defined in http://www.cms.gov/Medicare/Coding/HCPCSReleaseCodeSets
  • Include all codes defined in http://www.cms.gov/Medicare/Coding/ICD10
  • Include all codes defined in http://ada.org/cdt
  • Include all codes defined in http://loinc.org
